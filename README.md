# PAM Toolkit — Open-Source Privileged Access Automation
> Enterprise-grade PAM workflows · Demo-safe · 100% open-source

Zero-standing-privilege automation suite built across CrowdStrike, FortiOS, JumpServer, Teleport, and OPA.

### Live Contributions
| Tool           | Feature                            | PR / Status |
|----------------|------------------------------------|-------------|
| CrowdStrike    | Noisy hosts detection              | [#1391](https://github.com/CrowdStrike/falconpy/pull/1391) |
| CrowdStrike    | Session replay                     | [#1394](https://github.com/CrowdStrike/falconpy/pull/1394) |
| FortiOS        | Auto-block malicious IPs           | [#406](https://github.com/fortinet-ansible-dev/ansible-galaxy-fortios-collection/pull/406) |
| FortiOS        | External threat feed import        | [#407](https://github.com/fortinet-ansible-dev/ansible-galaxy-fortios-collection/pull/407) |
| JumpServer     | English SAML docs (promoted)       | [#16269](https://github.com/jumpserver/jumpserver/pull/16269) |
| JumpServer     | Threat feed integration            | [#16283](https://github.com/jumpserver/jumpserver/pull/16283) |
| JumpServer     | Session recording + alerting       | [#16290](https://github.com/jumpserver/jumpserver/pull/16290) |
| OPA            | PAM policy as code (root deny)     | [#8073](https://github.com/open-policy-agent/opa/pull/8073) |

### Tools Included
- Ansible playbooks
- OPA policies
- Python utilities
- Demo-safe (no real credentials needed)
